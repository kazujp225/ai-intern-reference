AI用語辞典

AIインターンで知っておきたい用語をわかりやすく解説

🧠
LLM
Large Language Model / 大規模言語モデル
モデル・学習

GPTやClaudeなどの大規模な自然言語処理モデル。大量のテキストデータで学習し、文章生成・要約・翻訳などを行う。

🔍
RAG
Retrieval-Augmented Generation / 検索拡張生成
技術・手法

外部データベースから情報を検索し、LLMの回答に組み込む技術。ハルシネーションを抑え、最新情報を反映できる。

🎯
ファインチューニング
Fine-tuning
モデル・学習

事前学習済みモデルを特定のタスクやドメインに合わせて追加学習すること。少量のデータで高精度なモデルを構築できる。

📝
プロンプトエンジニアリング
Prompt Engineering
技術・手法

AIに適切な指示（プロンプト）を与えて望ましい出力を得る技術。LLMの性能を最大限引き出す重要スキル。

⚡
Transformer
トランスフォーマー
モデル・学習

自己注意機構を用いたニューラルネットワーク。GPTやBERTなど現代のAIモデルの基盤アーキテクチャ。

⚙️
MLOps
Machine Learning Operations
インフラ・運用

機械学習モデルの開発・デプロイ・運用を効率化するプラクティス。CI/CDやモニタリングを含む。

💾
ベクトルDB
Vector Database
インフラ・運用

テキストや画像を数値ベクトルに変換して格納・検索するデータベース。RAGやセマンティック検索に不可欠。

🤖
エージェント
AI Agent
技術・手法

自律的にタスクを実行するAIシステム。ツール呼び出しや計画立案を行い、複雑な問題を解決する。

🎮
強化学習
Reinforcement Learning
モデル・学習

試行錯誤を通じて報酬を最大化する学習手法。ゲームAIやロボット制御に活用される。

🏆
Kaggle
カグル
キャリア

データサイエンスのコンペティションプラットフォーム。実践的なスキルを磨き、ポートフォリオを構築できる。
